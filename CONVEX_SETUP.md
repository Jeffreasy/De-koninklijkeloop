# Convex Setup Required
The automated agents cannot complete the interactive Convex login/project selection.

## Action Required
Please run the following command in your terminal:
```bash
npx convex dev
```
1. It will ask to log in (browser).
2. Select standard options (e.g., "Existing Project" -> "frugal-goose-15").
3. This will generate the `convex/_generated` folder and fix the TypeScript errors.

Once done, the project is ready!
